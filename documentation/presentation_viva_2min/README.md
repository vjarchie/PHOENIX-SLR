# Presentation Pack (2-3 min Viva)

## Files in this folder

- `SLIDES_DOCUMENT.md` - final slide-wise content with likely examiner questions and safe answers.
- `SPEAKER_SCRIPT_2_3_MIN.md` - full timed script for delivery.
- `motivation_accessibility.svg` - visual for motivation slide.
- `hybrid_architecture.svg` - visual for architecture slide.
- `limitations_future.svg` - visual for limitations/future slides.

## Suggested slide build flow

1. Create 5 slides in PPT/Canva/Google Slides.
2. Copy slide text from `SLIDES_DOCUMENT.md`.
3. Insert SVG visuals from this folder:
   - Slide 1: `motivation_accessibility.svg`
   - Slide 2 or 3: `hybrid_architecture.svg`
   - Slide 4 and/or 5: `limitations_future.svg`
4. Practice with `SPEAKER_SCRIPT_2_3_MIN.md` and keep total under 3 minutes.

## Notes

- SVG files are fully scalable and editable in most presentation tools.
- Keep bullets short during delivery and rely on script wording for detail.
