\section{Development Environment and Hardware Constraints}

The system configurations deployed a Python 3.10 framework utilizing the highly optimized PyTorch 2.0 automatic differentiation engine. Deep continuous video arrays inherently generate massive unrolled computational graphs during backpropagation, placing immense stress on hardware memory boundaries. 

Due to strict hardware resource constraints, specifically a highly limited GPU Video RAM (VRAM) footprint, the fundamental input batch size was mathematically capped at an absolute maximum of 4 concurrent sequence elements. Furthermore, the input temporal volume was artificially bounded to a ceiling restriction of precisely $T=64$ contiguous frames. Exceeding this boundary reliably triggered deterministic out-of-memory cascading failures during the transformer self-attention calculations. 

The fully initialized `HybridCTCAttentionModel`, coupling the ResNet-18 extraction backend together with the 6-layer encoding and 3-layer autoregressive decoding array, culminated at 45.2 million distinct trainable parameters. Baseline network training procedures spanned approximately 20 continuous operational hours across 100 epoch evaluations.

\section{Data Augmentation Constraint Algorithms}

The aggressive model complexity relative to the restricted 5,000-sample dataset fundamentally necessitated complex auxiliary external augmentation procedures. To artificially inflate gradient variability, the `VideoAugmentation` and `TemporalSampling` functional logic classes dynamically manipulated each raw video batch upon extraction.

Given the unique semantic sensitivities inherent in accurate sign language representations, aggressive normalization metrics were specifically engineered:
\begin{enumerate}
    \item \textbf{Spatial Deformation Boundaries}: Target frames suffered dynamic Random Crop scaling strictly constrained mathematically to oscillate between factors of 0.85 and 1.0. Minor angular rotational shifts were enforced within extreme $\pm 8^{\circ}$ variance margins to simulate minor standing height discrepancies without structurally invalidating absolute spatial coordinates.
    \item \textbf{Color Jitter Dynamics}: Visual brightness, contrast, and saturation factors executed sequential shifting parameters constrained to $\pm$ 0.15 variances targeting systemic lighting independence.
    \item \textbf{Temporal Processing Matrix}: Recurrent sequences experienced stochastic speed perturbations oscillating heavily between $0.8\times$ and $1.2\times$ native runtime speeds via fixed linear interpolation. Additionally, aggressive temporal masking ratios, analogous to the acoustic SpecAugment paradigm, completely zero-padded 8\% of adjacent sequence inputs featuring a steady 15\% occurrence probability, directly simulating intermittent target occlusions.
    \item \textbf{Directional Preservation Law}: Distinctively, to definitively preserve critical spatial semantics (e.g., differentiating right-handed signs from left-handed counterparts), the continuous processing pipeline explicitly restricted any form of lateral horizontal planar flipping transformations entirely. 
\end{enumerate}

\section{Training Optimization Dynamics}

The network training parameters were systematically extracted and managed by the highly standardized `train\_e2e.py` orchestration runtime. All computational branches mathematically progressed employing the `AdamW` optimization module algorithm initialized against an aggressive weight decay factor defining a 0.05 absolute regularization strength.

The base learning rate logic utilized uniquely decoupled configuration matrices: the heavily pretrained visual ResNet-18 array tracked a restricted gradient limit scaling operating strictly at $1 \times 10^{-5}$, whereas the subsequent uninitialized transformer modeling layers pursued convergence curves spanning orders up to $1 \times 10^{-4}$. The scheduler implemented a deliberate 5-epoch linear warmup progression instantly mitigating volatile starting gradients, subsequently locking into a steady cosine decay annealing schedule down to practical zero variables over the 100-epoch cycle length. Absolute gradient explosions inherent to prolonged sequence modeling were forcibly managed by a firm maximum norm clamp equal to 5.0 parameter distances.
